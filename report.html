Project Report.


Website Requirements:

1. Visitor Interaction.

Req Description Comments
1.1 The user will be able to quickly return to the home page from any other page.
1.2 The user will be able to subscribe to a mailing list.
1.3 Users will be able to quickly navigate the website from the top navbar.

2. Editing, Updates and Administration.

Req Description Comments
2.1 The user will be able to acces a constantly updated feed of the latest jailbreaks and tweaks.
2.2 The website will be updated automatically using other sites.
2.3 The users will be directly able to suggest changes/updates to existing articles as well as suggest new ones.

3. Sitemap and Navigation.

Req Description Comments
3.1 The top navbar will always be visible on all pages to allow quick navigation.
3.2 The user will be able to search the website using the search function availiable in the top navigation bar.
3.3 The user can go directly to different webpages through links and images on the website.

4. Accessibility.

Req Description Comments
4.1 The website uses a primarily black and white colour scheme which with high contrast making it easy to read for anyone.
4.2 Made using responsive html and css which allows it to be used on any screen without compromising usability.
4.3 Scrolling down the page will reveal and upwards facing arrow that takes the user back to the top of the page.

5. Styling and Design.

Req Description Comments
5.1 The website is simple,yet it stand out with high contrasting text and images.
5.2 On the home page the articles flow over the bottom of the screen, encouraging the user to scroll.
5.3 Images are animated, inviting the user to press on them.
